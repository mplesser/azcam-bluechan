# azcam-bluechan

## Purpose

This repository contains the *azcam-bluechan* *azcam* environment.  It contains the code and data files for the MMTO Spectrograph Blue Channel camera system.

## Installation

Download code and install python package:

```python
git clone https://github.com/mplesser/azcam-bluechan
pip install -r ./azcam-bluechan/requirements.txt
```

The `support` subfolder contains files and folders which may be useful in `datafolder`.  See `server_bluechan.py` and `console_bluechan.py`.

For Windows systems a common root folder would be `c:\azcam\azcam-bluechan`.
